Supported platforms & standards

We support a range of platforms and open standards to ensure seamless integration.

ArcGIS (Online/ Enterprise)

  • Feature Services, Hosted Layers, Web Maps
  • GeoJSON export and web map embedding
  • Single Sign-On (SSO): SAML / OAuth 2.0

MapGuide (Open Source / Autodesk)

  • Layer management, Fusion Viewer
  • Web Map Service (WMS) rendering for CARL embeds
  • Public/private layer control

External GIS / OGC

  • WMS / WFS / WMTS, XYZ tiles
  • GeoServer, QGIS Server compatible
  • GeoJSON & CSV for data exchange

Standards

  • Coordinate Systems: EPSG:4326 (WGS84), EPSG:3857 (Web Mercator), and regional systems (e.g., GDA2020).
  • Identity & Security: SSO via SAML 2.0 / OAuth 2.0 / OIDC, Role-Based Access Control (RBAC), audit logs, and least-privilege tokens.

Integration patterns

Our integration patterns enable a flexible and powerful connection between CARL and your GIS.

  • Embedded Map in CARL

    Display ArcGIS/MapGuide or external WMS/WFS inside CARL records and dashboards.

  • Deep Links from Map to CARL

    Click a feature to open the Asset, Location, or WO via Feature ID or custom keys.

  • Data Sync (one-way or two-way)

    Periodic or event-driven sync for geometry and attributes (status, access notes, ownership).

  • Event/Webhook Triggers

    GIS events (e.g., threshold within a polygon) create Work Requests/Orders with map snapshots.

  • Mobile & Offline

    CARL Flash uses offline basemaps and cached layers; queued sync resumes when network returns.

Core capabilities

  • Asset geocoding & validation: Store coordinates with accuracy source; validate against site boundaries.
  • Work order mapping: Draw polygons, lines, or points, attach redlines, and store extents on each WO.
  • Proximity & network analysis: Find the nearest depot, crew, or spare part; calculate drive-time polygons and response radius.
  • Condition overlays: Layer live meter points and alarms on the map; drill into trends directly from the WO.
  • Bulk actions: Select multiple features to raise batch WOs or schedule PMs across a corridor or zone.
  • Security & governance: RBAC, GIS group mapping, least-privilege tokens, IP allow-lists, and a full audit trail.
